# PID Departures for Home Assistant

Home Assistant custom integration for Prague PID departures using Golemio API.

## Features

- Next 5 departures
- Served lines
- Config Flow support
- Multiple stop instances
- HACS compatible

## Installation via HACS

1. Open HACS
2. Integrations
3. Three dots menu
4. Custom repositories
5. Add repository URL
6. Category: Integration
7. Install "PID Departures"

## Configuration

After installation:

Settings → Devices & Services → Add Integration

## Requirements

- Golemio API key
- PID stop ID
